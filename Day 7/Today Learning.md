## Day 7 - MongoDB CRUD Operations

### Topics Covered
- Connecting the Express server to MongoDB
- Creating a Notes collection in the database
- Storing notes in MongoDB
- GET API with MongoDB
- POST API with MongoDB

### Learned
- Connected my local Express server to a MongoDB database.
- Created a Notes collection and stored data in the database.
- Implemented GET and POST APIs to retrieve and add notes from/to MongoDB.
- Understood how APIs interact with a database instead of using in-memory data.

### Key Takeaway
Today I learned how to connect an Express server to a MongoDB database and perform GET and POST operations on data stored in the database. This helped me understand how backend applications communicate with a real database.
