## Day 9 - Full Stack CRUD Integration

### Topics Covered
- MongoDB CRUD Operations
- Integrating Frontend with Backend
- Using Axios for API Requests
- Fetching and Displaying Data
- Creating Notes through the Frontend

### Learned
- Performed complete CRUD operations on a MongoDB database.
- Built both the frontend and backend for the application.
- Used Axios on the frontend to make HTTP requests to the backend APIs.
- Connected the frontend with the backend to create, read, update, and delete notes.
- Successfully fetched data from the backend and displayed it dynamically on the frontend.

### Key Takeaway
Today I learned how a frontend and backend communicate with each other using APIs. By using Axios to connect the frontend with my Express backend, I was able to perform complete CRUD operations on a MongoDB database and display real-time data in the user interface.
